# (57)摘要

本发明公开了基于多传感器数据融合的电机状态监测方法、系统及设备，涉及工业监控技术领域，该方法包括：在电机壳体关键位置部署柔性应变-温度复合传感器阵列，实时采集获得多源融合信号，进行小波包变换，提取非平稳故障指纹并计算齿轮磨损拓扑不变量，与云端知识库进行增量式参数交换，结合实时工况动态生成健康度置信椭圆，若健康度超阈值，触发类脑决策单元进行仿真容错控制。本发明解决了现有电机状态监测方法因感知数据单一、特征提取浅层及决策机制固化，对早期复合故障的诊断灵敏度不足且容错控制滞后的技术问题，达到了多物理场博弈融合深层故障指纹与云端增量演化，实现对潜在故障的前瞻性识别和实时容错控制的技术效果。

# 背景技术

[0002] 随着工业自动化的发展，电机作为重要的动力设备广泛应用于各类机械系统中。然而，电机在长期运行过程中，容易受到磨损、过载、温度升高等因素的影响，导致性能衰退或故障。传统的电机状态监测方法通常依赖单一的传感器数据或浅层特征提取，难以全面、准确地反映电机的健康状态，尤其对于早期复合故障的诊断存在较低的灵敏度。此外，现有技术中，故障诊断与容错控制往往是独立进行，缺乏实时性与智能化，难以在故障发生时及时采取有效的容错措施，导致电机停机和维护成本增加。

[0015] 具体的，为实现对电机运行状态的全面监测，首先在电机壳体的关键位置部署柔性应变-温度复合传感器阵列。这些关键位置包括齿轮箱与电机壳体的连接处、电机轴的支撑点等，对电机性能和故障诊断具有决定性影响。此传感器阵列采用柔性材料设计，能够适应电机壳体的形变并实时采集各种物理信号，在复杂的工作环境中保持稳定性，提供可靠的数据支持。

[0016] 具体而言，该传感器阵列能够实时采集多源传感数据，这些数据包括齿轮箱的壳体形变数据、润滑油温度数据、高频振动信号以及三相电流数据。其中，齿轮箱的壳体形变数据是通过在齿轮箱壳体上安装的光纤光栅传感器来采集的。光纤光栅传感器基于光纤布拉格光栅(FBG)原理，能够高精度地监测壳体的微小形变。当齿轮箱内的齿轮在啮合过程中出现磨损、缺油或轴不对中等问题时，齿轮箱壳体的应力分布会发生变化，从而导致形变。光纤光栅传感器能够将这些形变转换为光信号，并通过解调器将其转换为电信号，进而实现对齿轮箱健康状态的实时监测。

[0017] 同时, 润滑油温度数据可利用红外油温传感器进行采集。红外油温传感器基于红外热成像原理, 能够非接触式地测量润滑油的温度。润滑油的温度是反映齿轮箱润滑状态的重要指标。如果润滑油温度过高, 可能意味着润滑油不足或润滑系统出现故障。通过实时监测润滑油温度, 可以提前发现潜在的润滑问题, 避免因润滑不良导致的齿轮磨损。

[0046] 最后,将这些量化的拓扑特征进行标准化处理,使其成为一个统一的健康度指数,即齿轮磨损拓扑不变量,便于与其他诊断指标进行比较和综合分析。生成的齿轮磨损拓扑不变量代表了电机健康状态中与齿轮磨损相关的稳定拓扑特征,不仅能够反映当前电机的磨损情况,还能作为未来故障预测的依据。通过对这些不变量的实时监测,可以提前发现电机潜在的故障问题,并及时采取必要的维护措施。
