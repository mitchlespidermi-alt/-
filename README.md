针对工业泵房场景中水泵驱动电机及其配电系统故障定位困难、人工排查效率低、非计划停机影响供水连续性等问题，本文面向“工业控制智能应用”赛题中的设备故障预警方向，提出一种基于多源传感融合的泵房设备故障预警系统方案。该方案以振动、温度和电流等传感器数据为核心输入，结合泵房典型故障机理，构建“传感采集—数据处理—故障分类—趋势预测—分级预警—控制联动—HMI显示”的整体架构。算法层采用 SVM 作为基线分类模型、随机森林作为主分类模型、LSTM 作为趋势预测模型，以实现对电机绕组开路、绕组不平衡与匝间短路、水泵堵转及连接机构断开等异常状态的识别与提前预警；系统层采用软件侧与控制侧分层设计，软件侧完成数据分析与模型推理，并通过 MQTT 与 EAE 平台交换数据。
