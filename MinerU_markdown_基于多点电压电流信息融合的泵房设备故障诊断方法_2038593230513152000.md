# A Method for Fault Diagnosis of Equipment in Pumping House Based on Multipoint Voltage and Current Information Fusion

MENG Zhiqiang，CHEN Liqin† ，CHEN Yandong，LUO Jun 

（College of Electrical and Information Engineering， Hunan University， Changsha 410082， China） 

Abstract：To realize the rapid online diagnosis and alarm of the faults of power distribution equipment and water pumps in the pumping house of the water plant and its secondary booster pumping house so as to guarantee the safe operation of the pumping house system， an online diagnosis method of equipment in the pumping house is proposed based on the multipoint voltage and current information fusion. Based on the distribution equipment and pumps in the pumping house， the voltage and current characteristics of common faults in the power distribution equipment and water pumps in the pumping house， as well as the mutual influence mechanism of these faults， are systematically analyzed for the first time. The real-time voltage and current values of multiple key nodes in the distribution line are fused to extract the fault characteristics of six types of faults， including voltage overruns in the main circuit of the pumping house distribution， open air switches or contactors in the distribution line， open pump 

motor windings， imbalanced pump motor winding， inter-turn short circuit， pump blocking， connection break between the motor and pump， so as to realize the online diagnoses of these six types of faults. The implementation conditions of the method， the principle of implementation， and the specific steps are elaborated. Finally， MATLAB/ Simulink is used to build a simulation model， which is experimentally verified. The simulation results show that the method can effectively diagnose the pumping house equipment faults， overcome the shortcomings of time-consuming and untimely offline analysis in the labor-intensive manual diagnosis， and provide effective protection for the safe operation of the distribution equipment and pumps in the water plant and its secondary booster pumping house. 

Key words：power distribution equipment；water pump motor；fault diagnosis；information fusion

# 2.1 供电电压偏差与超限

# 2.1.1 供电电压偏差与平衡计算

《电能质量 三相电压不平衡》（GB/T 15543—2008）定义三相电压不平衡为：三相电压在幅值上不同或相位差不是 $1 2 0 ^ { \circ }$ ，或兼而有之［13］ .本文主要考虑三相电压幅值上的不同，可以使用电压偏差 $\delta _ { \mathrm { v } i }$ 进行评价.

$$
\delta_ {\mathrm {V} i} = \frac {U _ {i} - U _ {\mathrm {N}}}{U _ {\mathrm {N}}} \times 100 \% \quad (i = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag{1}
$$

式中： $U _ { i }$ 为测量时间窗口（10个周波）内相电压测量值的平均值； $U _ { \mathrm { { N } } }$ 为系统标称相电压

若 $\delta _ { \mathrm { v } i }$ 满足：

$$
\delta_ {\mathrm {V} _ {- \min }} \leqslant \delta_ {\mathrm {V} i} \leqslant \delta_ {\mathrm {V} _ {- \max }} \quad (i = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {2}
$$

$$
\Delta V _ {i j} = \left| \delta_ {\mathrm {V} i} - \delta_ {\mathrm {V} j} \right| \leqslant \varepsilon_ {1} (i, j = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {3}
$$

则主回路供电电压是三相平衡的 $. { \mathcal { E } } _ { 1 }$ 为设定正小数；$\delta _ { \mathrm { v \_ m i n } } \cdot \delta _ { \mathrm { v \_ m a x } }$ 分别为设置的 $\delta _ { \mathrm { v } i }$ 最小值和最大值.

# 供电电压超限

若主回路电压偏差 $\delta _ { \mathrm { v } _ { i } }$ 满足：

$$
\delta_ {\mathrm {V} i} > \delta_ {\mathrm {V} _ {-} \max } > 0 \quad (i = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {4}
$$

则供电电压超过上限 若满足：

$$
\delta_ {\mathrm {V} i} <   \delta_ {\mathrm {V} _ {-} \min } <   0 \quad (i = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {5}
$$

则供电电压超过下限

在实际应用中，电压偏差限值 $( \delta _ { \mathrm { v \_ m i n } } \delta _ { \mathrm { v \_ m a x } } )$ ）可以根据设备和电网实际进行合理设置，作为供电电压超限故障判断的阈值

# 2.2 电流不平衡率

在三相供电电压平衡条件下，电流不平衡率 $\delta _ { { \scriptscriptstyle \mathrm { I } i } }$ 又称为用电负荷不平衡率，是描述三相电流幅值不相等程度和衡量设备是否发生过流故障的一个重要技术指标，一般使用绝对值定义：

$$
\delta_ {1 i} = \left| \frac {\bar {I} - I _ {i}}{\bar {I}} \right| \times 100 \% \quad (i = a, b, c) \tag{6}
$$

$$
\bar {I} = \frac {I _ {\mathrm {a}} + I _ {\mathrm {b}} + I _ {\mathrm {c}}}{3} \tag {7}
$$

式中： $I _ { i } ( i { = a } , \mathrm { b } , \mathrm { c } )$ 为相电流；-I为三相电流平均值.

若某节点处的三相电流不平衡率 $\delta _ { { \scriptscriptstyle \mathrm { I } i } }$ 满足：

$$
\delta_ {1 i} \geqslant \delta_ {1 \max } \quad (i = a, b, c) \tag {8}
$$

则表明该节点后的水泵电机发生了重载过流故障，如水泵堵转或电机绕组匝间短路. $\delta _ { \mathrm { I m a x } }$ 为设定的正阈值.

# 2.3 配电线路中空气开关或接触器触头开路

由电机工作原理和图 可知，某一回路电机的三相输入电压和输入电流满足：

$$
U _ {1 i} = 0 \quad (i = a, b, c) \tag {9}
$$

$$
I _ {i} = 0 \quad (i = a, b, c) \tag {10}
$$

表明该回路空气开关或接触器的三相触头发生了开路 .

若回路空气开关或接触器的两相触头同时发生开路，则电机三相输入电流均为0，即满足式（10），由于此种情况下的电机定子绕组是一个等位体，故对应两开路相的端电压均等于未开路相输入端电压，且稳定.

若回路空气开关或接触器的某一相触头发生开路，使得电机缺相，由电机工作原理和图 可知：电机的对应相输入电流为0，即满足式（10），对应相输入端电压则等于另外两相输入线电压经 型等效电路（图2）的分压，明显低于正常输入端电压，使得电机三相端电压严重不平衡，即按式（ ）计算的电机对应相输入端电压偏差 $\delta _ { \mathrm { v } 1 i }$ 满足：

$$
\delta_ {\mathrm {V} 1 i} <   \delta_ {\mathrm {V} 1 _ {-} \min } <   \delta_ {\mathrm {V} _ {-} \min } <   0 \quad (i = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {11}
$$

式中： $\delta _ { \mathrm { v 1 \_ m i n } }$ 为电机端电压偏差设定值.这一判断方法同样适用于三相定子绕组星形连接的三相异步电机.

水泵电机绕组开路是指电机某一相定子绕组与电机输入端之间的连接断开 在电机三相输入端电压不为 的条件下，断开相的电流为 ，即满足式（ ）

# 2.5 水泵电机绕组不平衡与匝间短路

电机的单个绕组是由多匝绝缘线圈紧密叠合构成的 若线圈的绝缘破坏，将形成匝间短路，在电机三相输入端电压平衡条件下，匝间短路相的电流较大，使得电机三相绕组电流不平衡 即在电机三相端电压的电压偏差 $\delta _ { \mathrm { v } 1 i }$ 满足

$$
\delta_ {\mathrm {V} _ {- \min }} \leqslant \delta_ {\mathrm {V} 1 i} \leqslant \delta_ {\mathrm {V} _ {- \max }} \quad (i = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {12}
$$

$$
\Delta V _ {1 i j} = \left| \delta_ {\mathrm {V} 1 i} - \delta_ {\mathrm {V} 1 j} \right| \leqslant \varepsilon_ {1} \quad (i, j = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {13}
$$

条件下，按式（6）计算电机的三相电流不平衡率 $\delta _ { { \scriptscriptstyle { \mathrm { I } } i } }$ ，若满足：

$$
\Delta I _ {i j} = \left| \delta_ {\mathrm {I i}} - \delta_ {\mathrm {I j}} \right| > \varepsilon_ {2} (i, j = \mathrm {a}, \mathrm {b}, \mathrm {c}) \tag {14}
$$

则出现电机绕组不平衡. $\varepsilon _ { 2 }$ 为设定正小数.进一步比较 $\delta _ { { \scriptscriptstyle \mathrm { I } } i }$ ，寻找最大值 $\delta _ { \mathrm { { I m } } }$ .

$$
\delta_ {\mathrm {I m}} = \max  \left\{\delta_ {\mathrm {I a}}, \delta_ {\mathrm {f b}}, \delta_ {\mathrm {I c}} \right\} \tag {15}
$$

则 $\delta _ { \mathrm { { I m } } }$ 所对应的相电机绕组发生了匝间短路.

# 2.6 水泵堵转

当水泵出现堵转时，电机负荷加重，在三相输入端电压平衡条件下，电机三相电流平衡且明显增大.若电机端电压满足式（ ）、式（ ），且按式（）计算的电机电流平均值满足：

$$
\alpha I _ {\mathrm {N}} \leqslant \bar {I} \leqslant I _ {\max } \quad (\alpha > 1) \tag {16}
$$

则表明水泵发生了堵转 $. \alpha$ 为设定值， $I _ { \mathrm { { N } } }$ 为电机的额定电流， $I _ { \mathrm { m a x } }$ 为设定的电机保护动作电流.

# 2.7 电机与水泵连接机构断开

电机与水泵连接机构断开使得电机空载，水泵停止旋转，电机产生很小的空载相电流.

若电机端电压满足式（12）、式（13），且电机电流平均值满足：

$$
\bar {I} \leqslant \beta I _ {\mathrm {N}} \quad (\beta <   1) \tag {17}
$$

则表明电机与水泵连接机构断开. β为设定值.

# 5 结 论

本文对泵房配电系统中空气开关、接触器、电机等关键设备故障情况下回路中关键节点的电压、电流特征及其相互影响关系进行了分析.基于上述分析结果，得出了这些设备几类常见故障的诊断原理，提出了一种基于多点电压与电流信息融合分析的泵房设备故障诊断方法，详细阐述了该方法的执行条件和实现步骤 通过 仿真建模验证了故障诊断方法的可行性

本文提出的泵房设备故障诊断方法可用于实现自来水厂及其二次加压泵站配电系统设备与电机故障的实时诊断与定位，对泵房的安全供水具有重要的现实意义和工程应用价值，同时，该故障诊断方法具有通用性，可以应用于其他机电系统
